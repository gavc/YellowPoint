# YellowPoint

A lightweight Windows tray app that highlights the mouse pointer with a filled yellow circle.

## Features
- Toggle highlight with Ctrl+Alt+Y
- Adjustable size and opacity
- Runs in the system tray

## Run
- Build and run from Visual Studio or with `dotnet run`

## Notes
- Settings are stored in %AppData%\YellowPoint\settings.json
- Icons from: https://www.flaticon.com/
